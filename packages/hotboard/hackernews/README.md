# Hotboard Hacker News

Hacker News 热榜数据获取工具。

## 安装

```bash
pip install hotboard-hackernews
```

## 使用

```bash
# 获取热榜（Markdown 格式）
hotboard-hackernews

# 获取 JSON 格式
hotboard-hackernews --format json

# 查看帮助
hotboard-hackernews --help
```

## 输出格式

- **Markdown**: 适合阅读的格式化输出
- **JSON**: 结构化数据，便于程序处理
